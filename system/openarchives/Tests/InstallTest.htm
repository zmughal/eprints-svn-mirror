Testing an Open Archives Software Installation

This is a list of links to test an Open Archives Software Installation.  Change the string host:port to the actual host name and port of your server.

The following URLs should return basic information about your server:

http://dibble.ecs.soton.ac.uk:80/Dienst/Info/1.0/Identity

http://dibble.ecs.soton.ac.uk:80/Dienst/Info/3.0/List-Services

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/2.0/List-Verbs

The following URLs should return a simple stub XML response for the verb.  Once you write the link from the Open Archives Software to your archive, these URLs can then be used to test your link (you will of course need to change the identifier in the requests).

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/4.0/List-Contents

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/1.0/Disseminate/handlecorp/970101/%23oams/xml

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/1.0/List-Meta-Formats

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/2.0/List-Partitions

http://dibble.ecs.soton.ac.uk:80/Dienst/Repository/1.0/Structure/handlecorp/970101?view=%23
