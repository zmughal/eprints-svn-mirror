Welcome to RobPrints, the experimental implementation of the EPrints software, currently under development by Robert Tansley. Please select an option from below:

Browse the archive by subject

This option lets you browse the subject hierarchy and see what papers are in each subject category.

Search the archive

Search the archive. A simple search will let you search for author, title, publication and year, and advanced search will let you search more fields too.

Register to become a RobPrints user

If you don't yet have a RobPrints user account, select this option. This page lets you subscribe to the RobPrints E-mail update service, or register as a RobPrints author.

Enter your RobPrints author area

Select this option to access your RobPrints author area. From this area you can update your personal details, submit new papers, and remove and update existing papers from the archive.

Password Reminder

If you've forgotten your username or password, then the system can remind you.
