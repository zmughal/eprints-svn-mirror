Copyright (C) 2000, University of Southampton

L'université de Southampton détient les droits de reproduction sur les composants principaux de ce système, à l'exception du composant des archives ouvertes (dans le répertoire openarchives/), qui est une version modifiée du code distribué par Cornell University Digital Library Research Group.

Ce logiciel est distribué gratuitement. Les versions modifiées de ce logiciel peuvent être distribuées gratuitement à condition qu'un fichier README soit inclus décrivant les modifications et l'adresse où se procurer la version originale.

Ce logiciel est fourni sans garant d'adéquation au but recherché. L'utilisation de ce logiciel se fait aux risques de l'utilisateur final.

IMPORTANT : Informez-nous des problèmes que vous rencontrez en vous référant à la liste des bogues eprints. Ce système comporte également une rubrique suggestion que vous pouvez utiliser pour nous faire des remarques ou des propositions d'amélioration du logiciel.

Adressez toute demande technique à :

support@eprints.org

Toute autre demande doit être adressée à :

info@eprints.org
